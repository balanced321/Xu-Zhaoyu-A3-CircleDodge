# Circle Dodge

## Overview
Circle Dodge is a simple 2D survival game made with the MohawkGame2D template.  
The player controls a blue circle and must avoid falling red enemies while collecting yellow stars for points.  
The game continues until the player collides with an enemy. The goal is to survive as long as possible and earn the highest score.

## How to Play
- Move using **WASD** or the **Arrow Keys**
- Avoid the **red falling enemies**
- Collect the **yellow stars** to increase your score
- If you touch an enemy, the game ends
- Press **SPACE** to restart after losing

## Objective
Try to collect as many stars as possible before getting hit.  
The game keeps track of your current score and your high score.

## Features
- Player movement with keyboard controls
- Falling enemy obstacles
- Collectible star objects
- Score and high score tracking
- Restart system after game over
- Simple background and UI

## Classes Used
This project uses the following user-defined classes:
- `Player`
- `FallingEnemy`
- `StarCollectible`

The game also uses an array of `FallingEnemy` objects with 10 elements.

## File Structure
- `Game.cs` - main game logic, setup, update loop, UI, and background
- `Player.cs` - player movement, drawing, and collision
- `FallingEnemy.cs` - enemy falling behavior and drawing
- `StarCollectible.cs` - collectible spawning and drawing

## Tools / Library
This project was created using the **MohawkGame2D** course template and only uses functions from the template library.
